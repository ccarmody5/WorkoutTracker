## 🎯 Quick Overview

**WorkoutTracker** is a Python-based fitness tracking application that helps you:
- Record workout activities and exercise sessions
- Track reps, weight, and workout duration
- Manage multiple user profiles
- Create and customize activity types
- View workout history and statistics

## 🛠️ Tech Stack

- **Backend:** Flask (Python web framework)
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy with Alembic migrations
- **Frontend:** HTML/CSS/JavaScript templates
- **Architecture:** Terminal-based CLI and web-based interface

## 🚀 Getting Started Quickly

1. **Install dependencies:** `pip install -r requirements.txt`
2. **Configure database:** Set up PostgreSQL and update your connection string
3. **Run migrations:** Execute Alembic migrations
4. **Start the application:**
   - Terminal: `python terminal_program.py`
   - Web: `python webapp.py`

## 📖 Choose Your Interface

### Terminal Interface (`terminal_program.py`)
- Interactive command-line workout tracking
- Menu-driven user experience
- Activity and user management

### Web Interface (`webapp.py`)
- Modern web-based UI
- REST API endpoints
- Browser-accessible at `http://localhost:5000`
